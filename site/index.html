    Go Starter Kit¶

    Un outil CLI puissant pour generer des projets Go prets pour la production en quelques secondes.

    Demarrage rapide¶

    Installation - Installez et creez votre premier projet en 30 secondes

    Guide d'utilisation - Structure de projet avec separation claire des responsabilites

    Projet genere - JWT, Docker, CI/CD, tests et plus inclus par defaut

    Contribuer - Contribuez au projet et ameliorez l'ecosysteme Go

    Apercu¶

    create-go-starter est un generateur de projets Go qui cree une architecture hexagonale complete avec toutes les fonctionnalites essentielles d'une application backend moderne. En une seule commande, obtenez un projet structure avec authentification JWT, API REST, base de donnees, tests, et configuration Docker prete pour le deploiement.

    Fonctionnalites incluses¶

    Fonctionnalite Description
    Architecture hexagonale Ports & Adapters - Separation claire des responsabilites
    Authentification JWT Access tokens + Refresh tokens avec rotation securisee
    API REST Framework Fiber v2 haute performance
    Base de donnees GORM avec PostgreSQL et migrations automatiques
    Injection de dependances uber-go/fx pour une architecture modulaire
    Tests complets Tests unitaires et d'integration inclus
    Documentation Swagger API documentee automatiquement avec OpenAPI
    Docker Build multi-stage optimise et docker-compose
    CI/CD Pipeline GitHub Actions pre-configure
    Logging structure rs/zerolog pour des logs professionnels

    Installation rapide¶

    go install github.com/tky0065/go-starter-kit/cmd/create-go-starter@latest

    git clone https://github.com/tky0065/go-starter-kit.git
cd go-starter-kit
go build -o create-go-starter ./cmd/create-go-starter

    Demarrage en 30 secondes¶

    # 1. Installer l'outil
go install github.com/tky0065/go-starter-kit/cmd/create-go-starter@latest

# 2. Creer un projet
create-go-starter mon-projet

# 3. Configuration automatique
cd mon-projet
./setup.sh

# 4. Lancer
make run

# 5. Tester
curl http://localhost:8080/health

    Stack technique¶

    Les projets generes utilisent les meilleures bibliotheques de l'ecosysteme Go :

    Composant Bibliotheque Description
    Web Fiber v2 Framework HTTP haute performance
    ORM GORM ORM avec support PostgreSQL
    DI fx Injection de dependances par Uber
    Logging zerolog Logger structure haute performance
    JWT golang-jwt v5 Tokens JWT
    Validation validator v10 Validation de structs
